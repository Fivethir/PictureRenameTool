# 图片重命名工具

一款专为设备管理设计的 Windows 图片重命名工具，支持自定义设备名称、备注、数量，并自动生成规范命名的副本文件，同时记录领取日志。

---

## ✅ 功能特点

- **图形化界面**：简洁直观，操作便捷
- **设备选择**：7 个预设设备（交换机、电池、水晶头、面板、模块、热熔设备、其他）
- **自定义设备**：选择“其他”可输入任意设备名称
- **备注支持**：可为设备添加备注（如“八口”、“备用”等）
- **数量输入**：输入领取数量（如 10），文件名自动包含该数字
- **智能命名**：
  - 有备注：`20251106_交换机_八口_10.jpg`
  - 无备注：`20251106_电池_5.jpg`
- **保留原图**：仅生成副本，不修改原始文件
- **设备统计**：左侧实时显示各设备累计领取总数
- **安全日志**：
  - 日志格式：`[2025-11-06 20:30:15] 领取交换机_八口_100个（文件：20251106_交换机_八口_100.jpg）`

---

## 📦 使用说明

### 运行方式（两种）

#### 方式一：直接运行 Python 脚本（需安装 Python）
1. 安装 [Python 3.6+](https://www.python.org/downloads/)
2. 将 `rename_picture.py` 保存到本地
3. 双击运行，或在终端执行：```bash python rename_picture.py
#### 方式二：运行独立 .exe（推荐，无需 Python）
1. 从 dist/图片重命名工具.exe 启动程序
2. 可复制到任意 Windows 电脑直接使用
